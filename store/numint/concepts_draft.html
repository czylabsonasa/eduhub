intro

we want to compute the value of the integral:

\int_{a}^{b} f(t)\text{d}t

where f:[a,b]\to {\mathbb R}.

when do we need to approximate integrals? If

  • it cannot be expressed in terms of elementary functions (e^{-x^2} for example).
  • the antiderivative (primitive function) is known, but too complex.
  • we should calculate a large number of “similar” type integrals.
  • the value of f is known only at a few points.

the approach

we are searching the approximation of \int_{a}^{b} f(t)\text{d}t in form of

I=\sum_{i=1}^n w_k f(t_k)

  • where t_1,\ldots ,t_n are the nodes (t_k\in [a,b]),
  • and w_1,\ldots ,w_n are the weights (they don’t depent on f).
  • I is called a quadrature rule (formula), its free parameters are n, t_1,\ldots ,t_n, w_1,\ldots ,w_n

interpolational quadrature formulas

  • more specifically: for a given f and nodes a=t_1,\ldots ,t_n=b and values f(t_1),\ldots,f(t_n) we determine the Lagrange interpolational polyinomial P and its exact integral value will be used to approximate \int_{a}^{b} f(t)\text{d}t.

    \begin{gather*} f(t) \approx P(t)\\ \int_{a}^{b} f(t)\text{d}t \approx \int_{a}^{b} P(t)\text{d}t \end{gather*}

  • we will learn what are the weights by using 1,2,3 nodes and how the corresponding quadrature rules are behaving.

midpoint (or tangent) method

  • one node interpolation
Importantsimple rule

I_{M}=(b-a)f\left(\frac{a+b}{2}\right)

  • to prove it just examine the plot
Noneplot

Noneerror

\left\vert \int_{a}^{b} f\text{d}t - I_{M}\right\vert \le \frac{M_2}{24}(b-a)^3

  • some details can be found here
Importantcompound rule
  • n nodes, n-1 intervals, mid-points
  • a=t_1<t_2<\ldots<t_n=b
  • the formula is for equally spaced nodes: t_k=a+(k-1)h with h=\frac{b-a}{n-1}
  • the mid-points: m_k=\frac{t_k+t_{k+1}}{2} for k=1,\ldots,n-1

I_{Mc}=h\left[f(m_1)+f(m_2)+\ldots+f(m_{k-1})\right]

Noneerror
  • n nodes, h=\frac{b-a}{n-1}

\left\vert \int_{a}^{b} f\text{d}t - I_{Mc}\right\vert \le (b-a)\frac{M_2}{24}h^2

  • by adding together the small-interval errors
Nonematlab 

% using n nodes compute the approx integral 
% of sin(t^2) on [a,b] w/ the mid-point method
clear;
fun=@(t) sin(t.^2);
a=0;
b=1;
n=ceil(3.2^2); % 3.2 sqrt(10)
t=linspace(a,b,n);
h=(b-a)/(n-1);
m=(t(1:n-1)+t(2:n))/2;
I=h*sum(fun(m))
E=abs(integral(fun,a,b)-I)

trapesoidal method

  • two nodes interpolation
Importantsimple rule

I_{T}=(b-a)\frac{f(a)+f(b)}{2}

  • to prove it just examine the plot
Noneplot

Noneerror

\left\vert \int_{a}^{b} f\text{d}t - I_{T}\right\vert \le \frac{M_2}{12}(b-a)^3

  • some details can be found here
Importantcompound rule
  • n nodes, n-1 intervals
  • a=t_1<t_2<\ldots<t_n=b
  • the formula is for equally spaced nodes: t_k=a+(k-1)h with h=\frac{b-a}{n-1}

I_{Tc}=\frac{h}{2}\left[f(t_1)+2f(t_2)+\ldots+2f(t_{n-1})+f(t_{n})\right]

Noneerror
  • n nodes, h=\frac{b-a}{n-1}

\left\vert \int_{a}^{b} f\text{d}t - I_{Tc}\right\vert \le (b-a)\frac{M_2}{12}h^2

  • by adding together the small-interval errors
Nonematlab 

% using n nodes compute the approx integral 
% of sin(t^2) on [a,b] w/ the trapesoidal method

clc; clear;
fun=@(t) sin(t.^2);
a=0;
b=1;
for k=1:5
  n=ceil(3.2^k); % 3.2 sqrt(10)
  t=linspace(a,b,n);
  h=(b-a)/(n-1);
  I=h/2*sum([fun(t(1)),2*fun(t(2:n-1)),fun(t(n))]);
  E=abs(integral(fun,a,b)-I)
end

Simpson method

  • three nodes interpolation
Importantsimple rule

I_{S}=(b-a)\frac{f(a)+4f(\frac{a+b}{2})+f(b)}{6}

  • compute the P(t) interpolational polynomial for the nodes a,\frac{a+b}{2},b and values f(a),f\left(\frac{a+b}{2}\right),f(b)
  • evaluate the integral \int_{a}^{b} P(t)\text{d}t
Noneplot

Noneerror
  • assuming that f^{(4)} is continuous on [a,b].

\left\vert \int_{a}^{b} f\text{d}t - I_{S,1}\right\vert \le \frac{M_4}{90\cdot 2^5}(b-a)^5

  • not so simple to get this…
Importantcompound rule
  • n nodes, n-1 intervals
  • a=t_1<t_2<\ldots<t_n=b
  • the formula is for equally spaced nodes: t_k=a+(k-1)h with h=\frac{b-a}{n-1}

I_{Sc}=\frac{1}{3}I_{Tc}+\frac{2}{3}I_{Mc}

Noneerror
  • n nodes, h=\frac{b-a}{n-1}

\left\vert \int_{a}^{b} f\text{d}t - I_{Sc}\right\vert \le (b-a)\frac{M_4}{90\cdot 2^5}h^4

  • by adding together the small-interval errors
Nonematlab 

% using n nodes compute the approx integral 
% of sin(t^2) on [a,b] w/ the Simpson method

clc; clear;
fun=@(t) sin(t.^2);
a=0;
b=1;
for k=2:5
  n=ceil(1.8^k); % 1.8 > 10^(1/4)
  t=linspace(a,b,n);
  h=(b-a)/(n-1);
  T=h/2*sum([fun(t(1)),2*fun(t(2:n-1)),fun(t(n))]);
  M=h*sum(fun((t(1:n-1)+t(2:n))/2));
  I=1/3*T+2/3*M;
  E=abs(integral(fun,a,b)-I)
end
