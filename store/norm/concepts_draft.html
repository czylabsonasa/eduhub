intro (abs)

  • in this section \(x,y \in \mathbb{R}\)
  • how large is a real number?
  • absolute value, distance from zero

\[ \vert x \vert = \begin{cases} x & \text{ if }\ \ x\ge 0\\ -x & \text{ otherwise } \end{cases} \]

  • properties:
    • \(\vert x\vert\ge 0\) and \(\vert x\vert=0\ \text{ iff }\ x=0\)
    • \(\vert x+y \vert \le \vert x\vert + \vert y \vert\)
    • \(\vert x\cdot y \vert = \vert x\vert \cdot \vert y \vert\)

review (vector norm)

  • in this section \(x,y \in \mathbb{R}^n\) and \(\alpha \in \mathbb{R}\)
  • how large is a vector?
  • it is denoted by \(\Vert x \Vert\)
  • requirements (axioms):
    • \(\Vert x\Vert\ge 0\) and \(\Vert x\Vert=0\ \text{ iff }\ x=0\)
    • \(\Vert x+y \Vert \le \Vert x\Vert + \Vert y \Vert\)
      • it is the triangle inequality
    • \(\Vert \alpha \cdot x \Vert = \vert \alpha\vert \cdot \Vert x \Vert\)
  • the following vector norms are important
    • \(\Vert x \Vert_{2} = \sqrt{\vert x_1\vert^2 + \ldots + \vert x_n\vert^2}\ \ \) (euclidean or 2-norm)
    • \(\Vert x \Vert_{1} = \vert x_1\vert + \ldots + \vert x_n\vert\ \ \) (manhattan or 1-norm)
    • \(\Vert x \Vert_{\infty} = \max_{i=k\ldots n}\vert x_k\vert\ \ \) (\(\infty\)-norm)
  • the distance between vectors: \(\Vert x-y\Vert\)
    • small: close to each other
    • large: far from each other

intro (matrix norm)

  • in this section \(A,B \in {\cal M}_{n\times n}\) and \(x \in \mathbb{R}^n\) and \(\alpha \in \mathbb{R}\)
  • how large is a matrix?
  • it is denoted by \(\Vert A \Vert\)
  • requirements (axioms):
    • \(\Vert A\Vert\ge 0\) and \(\Vert A\Vert=0\ \text{ iff }\ A=0\)
    • \(\Vert A+B \Vert \le \Vert A\Vert + \Vert B \Vert\ \ \) (the triangle inequality)
    • \(\Vert \alpha \cdot A \Vert = \vert \alpha\vert \cdot \Vert A \Vert\)
  • how can we define a matrix norm?
    • induced matrix norm, it is induced (determined) by a vector norm
    • let’s fix a vector norm \(\Vert . \Vert\)

\[ \Vert A \Vert = \max_{x\neq 0} \frac{\Vert Ax\Vert}{\Vert x\Vert} \]

  • one can read it: what is the largest relative change that is caused for a vector \(x\) multiplying it by \(A\)
  • it can be proved that the right hand side is well defined and the requirements are fulfilled
  • the following matrix norms are important
    • \(\Vert . \Vert_{1}\ \ \) 1-norm, column norm
    • \(\Vert . \Vert_{2}\ \ \) 2-norm, euclidean norm, spectral norm
    • \(\Vert . \Vert_{\infty}\ \ \) \(\infty\)-norm, row norm
  • the distance between matrices: \(\Vert A-B\Vert\)
    • small: close to each other
    • large: far from each other

matrix norm computational rules

  • \(a_{ij}\) is \(j\)-th element of the \(i\)-th row for a matrix \(A\)

1-norm

\[ \Vert A\Vert _1=\max\limits_{1\leq j\leq n}\sum\limits_{i=1}^n\vert a_{ij}\vert \]

  • for each column compute the sum of absolute values and take the maximal one
  • asdasd

2 norm

\[ \Vert A\Vert _2=\sqrt{\lambda _{\max} (A^{T}A)} \]

  • take the square root of the maximal absolute value eigenvalue of the matrix \((A^{T}A)\)

\(\infty\) norm

\[ \Vert A\Vert _\infty=\max_{1\leq i\leq n}\sum_{j=1}^n\vert a_{ij}\vert \]

  • for each row compute the sum of absolute values and take the maximal one

example

\[ A=\left[\begin{array}{rrr} -3&0&4\\ 1&-1&2\\ -2&1&-2 \end{array}\right] \]

  • \(\Vert A\Vert_{1}= 8\)
  • \(\Vert A\Vert_{\infty}= 7\)

matlab

  • the function calls norm(A,1), norm(A,2), norm(A,inf) can be used

example (sensitivity)

  • compare the solutions of the linear systems below \[ \left[ \begin{array}{rr} 1&1\\ 1&1.0001 \end{array} \right] \left[\begin{array}{r} x_1\\ x_2 \end{array}\right] = \left[\begin{array}{r} 2\\ 2 \end{array}\right] \]
    • \([x_1,x_2]^{T}=[2,0]\) \[ \left[ \begin{array}{rr} 1&1\\ 1&1.0001 \end{array} \right] \left[\begin{array}{r} x_1\\ x_2 \end{array}\right] = \left[\begin{array}{r} 2\\ 2.0001 \end{array}\right] \]
    • \([x_1,x_2]^{T}=[1,1]\)
  • large difference, despite the matrix is the same and the right hand sides are very close to each other (in vector norm!)
  • how can we explain this behavior?

inequality (error in the right hand side)

  • let the matrix \(A\) be invertible and fix a vector norm and the corresponding matrix norm
  • consider the linear systems: \[ Ax=b \ \ \ \ \ \ \ A(x+\Delta{x})=b+\Delta{b} \]
  • we’ll have different solutions for the systems, but how far they are from each other?
  • the following inequality holds: \[ \frac{\Vert \Delta{x}\Vert}{\Vert x\Vert} \le \Vert A\Vert \Vert A^{-1}\Vert \frac{\Vert \Delta{b}\Vert}{\Vert b\Vert} \]
  • the inequality is tight, no gap between the sides
  • read it: the relative error in the solution is in the worst case is \(\Vert A\Vert \Vert A^{-1}\Vert\) times the relative error in the right hand side
  • the quantity \(\Vert A\Vert \Vert A^{-1}\Vert\) is called the conditional number of \(A\)
    • \(\text{cond}(A)=\Vert A\Vert \Vert A^{-1}\Vert\) \[ x_{\text{relative error}} \le \text{cond}(A) b_{\text{relative error}} \]

matlab

  • the call cond(A,1), cond(A,2), cond(A,inf) can be used

properties of the conditional number

  • its value does depend on the norm used
  • \(\text{cond}(A) \geq 1\)
  • if \(A\) is orthogonal (\(A^{T}A=E\)), then \(\text{cond}_{2}(A)=1\)
  • \[ \left\vert\frac{\lambda_{\max}}{\lambda_{\min}}\right\vert\leq \text{cond}(A) \]
    • here \(\lambda_{\max}\) and \(\lambda_{\min}\) are the largest and smallest eigenvalues in absolute value of \(A\)
  • for \(c\neq 0\), \(\text{cond}(cA)=\text{cond}(A)\)
