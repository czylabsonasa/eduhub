elsőrendű lineáris homogén

Noney'=A(t)y

y=Ce^{\int A(t)\text{d}t} \ \ \ \ C\in\mathbb{R}

  • speciálisan, ha A(t)=A konstans:

y=Ce^{At} \ \ \ \ C\in\mathbb{R}

Noney'+10y=0

y=Ce^{-10t}\hspace{1cm} C\in\mathbb{R}

Noney'=\log(t)y

\begin{gather*} \int \log(t)\text{d}t=\log(t)t-\int 1\text{d}t=\\ =\log(t)t-t+C \ \ \ \ C\in\mathbb{R}\\ y=Ce^{t(\log(t)-1)} \ \ \ \ C\in\mathbb{R} \end{gather*}

Nonematlab 
>> syms y(t)
>> disp(int(log(t)))
t*(log(t) - 1)
 
>> disp(dsolve(diff(y)==y(t)*log(t)))
C1*exp(t*(log(t) - 1))

elsőrendű lineáris inhomogén

Noney'=A(t)y+B(t)
  • konstans variálása
  • a homogén általános megoldása: Ce^{\int A(t)\text{d}t}
    • ez konkrét feladatnál már nem integrálos alakban van
  • ebben a C-t egy ismeretlen C(t) függvénynek tekintjük és behelyettesítjük az inhomogénbe:

\begin{gather*} C'(t)e^{\int A(t)\text{d}t} + {\color{blue}C(t)e^{\int A(t)\text{d}t}A(t)} = {\color{blue}A(t)C(t)e^{\int A(t)\text{d}t}}+B(t)\\ C'(t) = B(t)e^{-\int A(t)\text{d}t}\\ \end{gather*}

  • a végén egy sima integrálás marad
Noney'=y+x+2

\begin{gather*} y=Ce^{x}\ \ \ \text{a homogén megoldása}\\ C=C(x)\ \ \ \text{konstans variálása}\\ C'(x)e^{x}+C(x)e^{x}=C(x)e^{x}+x+2\\ C'(x)=(x+2)e^{-x}\ \ \ \text{parciálisan integráljuk}\\ C(x)=-(x+2)e^{-x}+\int e^{-x}\text{d}x=-(x+3)e^{-x}+K\\ y=\left(-(x+3)e^{-x}+K\right)e^x\\ \end{gather*}

Nonematlab 
>> syms y(x)
>> disp(int((x+2)*exp(-x)))
-exp(-x)*(x + 3)
 
>> disp(dsolve(diff(y)==y(x)+x+2))
C1*exp(x) - x - 3
Nonexy'-2y=2x^4

\begin{gather*} y'=\frac{2}{x}y+2x^3\\ y=Ce^{\int \frac{2}{x}\text{d}x}=Ce^{2\log(x)}=Cx^2\ \ \ \text{a homogén megoldása}\\ C=C(x)\ \ \ \text{konstans variálása}\\ C'(x)x^2+C(x)2x=\frac{2}{x}C(x)x^2+2x^3\\ C'(x)=2x\\ C(x)=x^2+K\\ y=(x^2+K)x^2\\ \end{gather*}

>> disp(dsolve(diff(y)==2/x*y+2*x^3))

NoneAy''+By'+Cy=0

  • \lambda_1\neq \lambda_2 valósak y(t)=K_1 e^{\lambda_1 t}+K_2 e^{\lambda_2 t}

Bernoulli

Noney'+g(x)y+h(x)y^a=0\ \ a\neq 1
  • elsőrendű lineárisra vezetjük vissza

\begin{gather*} y'+g(x)y+h(x)y^a=0 \ \ \ \ \ [*(1-a)y^{-a}]\\ (1-a)y'y^{-a}+(1-a)g(x)y^{1-a}+(1-a)h(x)=0\ \ \ \ \ [z=y^{1-a}]\\ z'+(1-a)g(x)z+(1-a)h(x) \end{gather*}

Nonexy^2y'=x^2+y^3

\begin{gather*} xy^2y'-y^3-x^2=0\\ y'-\frac{1}{x}y-xy^{-2}=0\ \ \ \ a=-2,\ z=y^3\\ z'-\frac{3}{x}z-3x=0 \end{gather*}

  • konstans variálással

\begin{gather*} z'=\frac{3}{x}z+3x\\ z=Ce^{\int \frac{3}{x}\text{d}x}=Ce^{3\log(x)}=Cx^3\ \ \ \text{a homogén megoldása}\\ C=C(x)\ \ \ \text{konstans variálása}\\ C'(x)x^3+C(x)3x^2=\frac{3}{x}C(x)x^3+3x\\ C'(x)=\frac{3}{x^2}\\ C(x)=-\frac{3}{x}+K\\ z=(-\frac{3}{x}+K)x^3\\ y=\left((-\frac{3}{x}+K)x^3\right)^{\frac{1}{3}} \end{gather*}

magasabbrendű egyenlet -> első-rendű egyenletrendszer

Noney^{(4)}-y'=0

\begin{gather*} y_0=y\\ y_1=y'\\ y_2=y''\\ y_3=y'''\\ y_0'=y_1\\ y_1'=y_2\\ y_2'=y_3\\ y_3'=y_1\\ \end{gather*}

None2xy''+y'=0

\begin{gather*} y_0=y\\ y_1=y'\\ \hline{} y_0'=y_1\\ y_1'=-\frac{y_1}{2x}\\ \end{gather*}

Nonexy'''-y''-xy'+y=0

\begin{gather*} y_0=&y\\ y_1=&y'\\ y_2=&y''\\ \hline{} y_0'=&y_1\\ y_1'=&y_2\\ y_2'=&\frac{-y_0+xy_1+y_2}{x} \end{gather*}

fokozatos közelítés

Noney'=f(t,y)\ \ \ \Leftrightarrow \ \ y(t)=y(t_0)+\int_{t_0}^t f(s,y(s))\text{d}s
  • a jobboldalba belerakva y-t visszakapjuk azt \to fixpont
  • ha f “jó” tulajdonságú akkor a fixpont-iteráció bármilyen kezdő helyről (pont/függvény) indulva a fixponthoz (ami itt egy függvény) kerül egyre közelebb.
